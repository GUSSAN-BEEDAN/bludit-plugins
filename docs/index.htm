About Bludit

Bludit is a flat file CMS and does not require a database. The simplicity and yet strong flexibility makes Bludit a perfect CMS for blogs and websites when the complicated and extensive overhead like Wordpress is not needed.

Its lightweight nature does not only make it easy to install and use, but also makes it very fast. Even on low-powered web servers & web spaces, Bludit works performantly.

More information about Bludit, as well as access to themes and plugins can be found on the official Bludit homepage.

Contact3

Contact3 is a contact form for the Bludit v3.X CMS. With the plugin your visitors can contact you via email using a contact form.

The form can be visually customized to your needs and is easy to configure.

Current version of Contact3 for Bludit CMS: v2.1.1

Download GitHub

Features

  • Adds a contact form to a selected page
  • Customizable form template
  • Support for PHP mail() and SMTP.
  • GDPR compliant
  • Send copy of mail to user
  • Spam protection with:
    • Google reCPATCHA v2
    • hCPATCHA
    • Logical Question

ImageGallery

ImageGallery is an image gallery for the Bludit v3.X CMS. With this plugin you can extend your Bludit CMS with an image gallery.

The image gallery can be visually customized to your needs and is easy to configure.

With the "Lite Version" you can add images to one album. With the extended pro version you can manage multiple albums, add albums on pages and customize the lightbox colors.

Current version of ImageGallery for Bludit CMS: v1.3.0

Download Free Version Buy Pro Version ($5) GitHub

Features Free Version

  • Adds an image gallery on a page
  • Includes a lightbox
  • Images can be managed in the admin interface.
  • Thumbnails are generated automatically

Features Pro Version

  • All from Free Version

Docs

How to install a plugin

  • Download the plugin.
  • Unzip if necessary.
  • Copy the plugin folder with all the files inside /bl-plugins/ folder.
  • Log into your Bludit panel administrator, go to Settings -> Plugins, and activate your new plugin.

Contact3

  • All configurations are explained in the settings.
  • For an individual template just copy the files contact3.php and contact3.css from the plugin folder /layout into your own template directory and customize them.

Image Gallery

  • Image are managed in the admin interface.
  • All configurations are explained in the settings.
  • Thumbnails are generated automatically.
  • For an individual layout just copy the files imagegallery.php and imagegallery.css from the plugin folder /layout into your own template directory and customize them.
  • Pro version with multiple albums is in development.
